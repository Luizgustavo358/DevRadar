# DevRadar

### Sobre:

Aplicativo e Aplicação Web para cadastro de Devs.

### Como rodar

- Rodar o backend com: 

```sh
cd backend
yarn dev
```

- Rodar o web:

```sh
cd web
yarn start
```

- Rodar o Aplicativo Mobile:

```sh
cd mobile
yarn start
```

### Curso:

Semana Omnistack do site [RocketSeat](https://rocketseat.com.br/), ministrado por [Diego Fernandes](https://github.com/diego3g).
